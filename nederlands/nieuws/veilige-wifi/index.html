Een veilige wifi gebruiken

22/11 - Alleen beschikbaar als Nederlandstalig artikel

Wifi is plezant! Maar veilig? Dat ontek je hier

Wat is Wifi?

Wifi een draadloze internet verdeler.

Met wifi kun je zonder Enthernet (internet) kabel werken.

Wifi is handig als je van je bureau moet werken.

Hoe werkt dit

Wifi werkt met radiogolven.

Deze golven worden uitgezonden door een router.

Deze golven worden uitgezonden door een router.

Daardoor ontvangt Je computer, telefoon of tablet deze golven.

Die golven zijn je internet

Het verschil tussen Openbare Wifi en Gesloten WiFi

Als je wifi hebt kun je ze splitsen tussen 2 soorten. Namelijk: Open en Gesloten wifi.

Wat is veiliger? Open of gesloten wifi?

Gesloten wifi is veiliger den Open

Op open wifi kan iedereen zich inloggen Zonder Passwoord

Gesloten wifi moet je inloggen met een passwoord.

Sommige internet adapters kun je de apparaten beheren wie op jou wifi zit

Wat kan er gebeuren als ik op open wifi zit

Een hacker kan beter een cyber aanval plannen door de IP (adres van je computer) op te halen wat van jouw apparaat is

En zo kan de hacker bestanden overplaatsen dankzij de IP

Daarom is het veiliger om op gesloten wifi te zitten
